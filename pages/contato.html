Por Que Este Projeto Foi Criado

Este Gerador de Currículos foi desenvolvido como parte da atividade prática da disciplina de Fundamentos de Programação para Internet da UNIPAR EAD.

O objetivo principal foi proporcionar uma vivência real no desenvolvimento web, aproximando os alunos do mercado de trabalho através da criação de uma aplicação completa e funcional.

Objetivos de Aprendizado

  • Praticar as técnicas de programação aprendidas durante a matéria
  • Desenvolver capacidade analítica através da interpretação de requisitos
  • Aprender a pesquisar e implementar soluções como um desenvolvedor real
  • Integrar diferentes tecnologias (PHP, HTML, CSS, JavaScript, jQuery)
  • Compreender versionamento com Git e GitHub
  • Criar interfaces funcionais sem necessidade de ser designer
